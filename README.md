# NBC LMS 오픈 기념 이벤트 페이지

> 04/29기수 신규 LMS 오픈 기념 여름 수료 완주 이벤트 안내 페이지

## 🔗 배포 주소

**https://nbc-lms-open-event-page.vercel.app/**

## 📋 이벤트 개요

| 항목 | 내용 |
|------|------|
| 대상 | 04/29기수 수강생 전원 |
| 목적 | 신규 LMS 오픈 기념 수료 장려 |
| 테마 | 더운 여름, 시원한 학습 응원 선물 |

## 🎁 보상 구조

### Stage 1 · 기본 혜택
- **조건**: 이벤트 기간 내 진도율 80% 달성
- **혜택**: 메가커피 상품권 5,000원

### Stage 2 · 추가 혜택 (업그레이드)
- **조건**: 진도율 80% 달성 + 평가 점수 60점 이상 (과제 40% + 시험 60%)
- **혜택**: 배스킨라빈스 교환권 10,000원
- Stage 2 달성 시 Stage 1 혜택은 자동 업그레이드 (중복 지급 아님)

### 보너스 · 퍼펙트 이수 챌린지 연계
- 별도 운영 중인 퍼펙트 이수 챌린지 달성자는 배민 상품권 30,000원 + 배스킨라빈스 교환권 10,000원 동시 수령 가능
- 두 이벤트 동시 달성 시 최대 40,000원 상당 혜택

## ⚠️ 유의사항

- 이벤트 기간 내 조건을 충족한 수강생을 대상으로 진행
- 진도율 및 평가 점수는 LMS 기록 기준으로 확인
- Stage 2 혜택은 과제 40% + 시험 60% 합산 평가 점수 60점 이상 달성 시 지급
- 부정 응시, 대리 제출, 비정상 제출, 미완료 제출 확인 시 이벤트 대상 제외
- 이벤트 상품은 운영 상황에 따라 동급 상품으로 변경 가능
- 중복 수령 등으로 경품 합산 금액이 5만 원을 초과하는 경우, 제세공과금 처리를 위해 신분증 제출 요청 가능
- Stage 2 달성 시 Stage 1 혜택(메가커피)은 배스킨라빈스 교환권으로 대체 지급되며 중복 지급 없음
- 퍼펙트 이수 챌린지 추가 혜택은 해당 이벤트 조건을 별도 충족한 경우에만 지급
